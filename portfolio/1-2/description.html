This course is designed to introduce students to computer science concepts including the basics of computer hardware and software. Students will be introduced to programming concepts and will program original video games and programs using block-based programming languages such as Scratch (scratch.mit.edu) and text languages such as JavaScript using Code.org. Students will create a personal website using HTML and CSS. Students will learn some basics of Photoshop and Illustrator in order to create original content for their programs. Students will maintain a website portfolio demonstrating completed projects in this class.
